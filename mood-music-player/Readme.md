# 🎧 Mood-Based Music Player

A modern music player web app that plays songs based on your mood 🎭

##  Features

* Mood-based song selection
* Real MP3 player (Play / Pause / Next)
* Progress bar + Seek control
* Add to Favorites (saved in browser)
* Dynamic background UI
* Favorite songs section

##  Tech Stack

* HTML
* CSS (Glass UI + Animations)
* JavaScript (DOM, LocalStorage, Audio API)

##  Folder Structure

```
mood-music-player/
│── index.html
│── style.css
│── script.js
│── assets/
│   └── songs/
│       ├── song1.mp3
│       ├── song2.mp3
│       └── song3.mp3
```

## ▶ How to Run

1. Download or clone the repo
2. Open `index.html` in browser
3. Select mood and enjoy 🎶
